# Player Agent Profiles

Defines the simulated player personas used in each testing round. Each agent approaches the game from a different perspective, ensuring broad coverage of the player experience.

## Agent 1: Casual Mobile Gamer

- **Name:** Casey
- **Age range:** 25-40
- **Skill level:** Moderate
- **Device comfort:** High — uses phone constantly, expects polished mobile UX
- **Goals:** Quick entertainment during a road trip. Wants to tap, see something fun happen, and not think too hard.
- **Evaluation focus:** Onboarding clarity, tap responsiveness, visual appeal, session flow (start/pause/resume)
- **Patience level:** Medium — will try a few things but abandons if confused
- **Key questions:** Can I figure out how to play within 10 seconds? Does tapping feel good? Is the game fun at a glance?

## Agent 2: Young Child

- **Name:** Milo
- **Age range:** 5-8
- **Skill level:** Low
- **Device comfort:** Low — borrows parent's phone, may tap randomly
- **Goals:** See cows! Tap things! Fun colors and sounds!
- **Evaluation focus:** Visual clarity of cows/graveyards, accidental input handling, large tap targets, delight factor
- **Patience level:** Very low — needs immediate feedback or loses interest
- **Key questions:** Are the cows obvious and delightful? Can a small child tap the right things? Is there enough visual reward for each action?

## Agent 3: Accessibility-Focused User

- **Name:** Avery
- **Age range:** 30-50
- **Skill level:** Moderate
- **Device comfort:** Moderate — may use assistive features (large text, high contrast, screen reader)
- **Goals:** Play the game comfortably with visual or motor accessibility needs.
- **Evaluation focus:** Contrast ratios, touch target sizes, text readability, color-blind safety, motion sensitivity
- **Patience level:** High — willing to explore, but frustrated by inaccessible UI
- **Key questions:** Can I read all text? Are touch targets large enough? Does the game rely solely on color to convey information?

## Agent 4: First-Time Player

- **Name:** Jordan
- **Age range:** 18-30
- **Skill level:** Low (for this game)
- **Device comfort:** High
- **Goals:** Understand the game concept and rules without prior knowledge of the road trip game.
- **Evaluation focus:** Onboarding/tutorial, discoverability of mechanics (tap cow, graveyard penalty), scoring clarity
- **Patience level:** Medium — willing to explore but expects the game to teach itself
- **Key questions:** Do I understand what to do without external explanation? Is the cow/graveyard scoring mechanic clear? Can I figure out which side is mine?

## Agent 5: Impatient Commuter

- **Name:** Reese
- **Age range:** 20-35
- **Skill level:** Moderate-High
- **Device comfort:** High — power user, fast swipes, expects instant response
- **Goals:** Kill time quickly. Wants to start playing in under 3 seconds. Zero tolerance for friction.
- **Evaluation focus:** Launch-to-gameplay speed, menu navigation efficiency, pause/resume reliability, perceived performance
- **Patience level:** Very low — any delay, stutter, or extra tap is a negative
- **Key questions:** How many taps from launch to gameplay? Does anything lag or stutter? Can I pause and resume seamlessly?

## Agent 6: Genre Enthusiast

- **Name:** Dakota
- **Age range:** 25-45
- **Skill level:** High
- **Device comfort:** High
- **Goals:** Evaluate this as a proper game — expects depth, polish, and satisfying game feel even in a simple concept.
- **Evaluation focus:** Game feel (juice, feedback, animations), visual consistency, scoring balance, replayability
- **Patience level:** High — will thoroughly explore all mechanics
- **Key questions:** Does the game have satisfying juice/feedback? Are animations smooth and consistent? Does the visual style hold together? Is there enough to keep me engaged across multiple trips?

## Agent 7: Backseat Parent

- **Name:** Morgan
- **Age range:** 35-50
- **Skill level:** Low-Moderate
- **Device comfort:** Moderate
- **Goals:** Set up the game for kids in the backseat, then occasionally glance at scores. Needs it to "just work" without intervention.
- **Evaluation focus:** Setup simplicity, score visibility at a glance, child-proofing (no accidental exits, no in-app purchases), trip persistence
- **Patience level:** Low for setup, high for passive monitoring
- **Key questions:** Can I hand this to my kid without explaining it? Can I see the score from the front seat? Does it save progress if the app is backgrounded?
