# Evaluation Rubric

Used by each player agent to evaluate the game consistently across testing rounds. Agents rate each category and provide free-text observations.

## Rating Scale

| Score | Label | Meaning |
|-------|-------|---------|
| 1 | Poor | Broken, confusing, or unacceptable |
| 2 | Below Average | Noticeable problems that hurt the experience |
| 3 | Acceptable | Works but has clear room for improvement |
| 4 | Good | Solid experience with minor issues |
| 5 | Excellent | Polished, delightful, no issues noticed |
| N/A | Not Applicable | Agent's persona cannot evaluate this area |

## Evaluation Categories

### 1. Core Gameplay Loop

Rate the fundamental play cycle: spot cow/graveyard, tap, see score update.

- **Clarity:** Is it obvious what to do and how scoring works?
- **Satisfaction:** Does the core loop feel fun and rewarding?
- **Pacing:** Does the game flow well during a real road trip scenario?
