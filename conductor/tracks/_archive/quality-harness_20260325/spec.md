## Summary

Integrate the tea-leaves Godot AI quality harness into the project, providing runtime DevTools (command server, screenshots, input simulation, state inspection), SceneValidator, and automated linting tools (project UID/NodePath, shader compilation, test conventions) — adapted to the CowsGraveyards namespace and .NET 9 stack.

## Context

Cows & Graveyards is developed with Claude Code as the primary AI assistant. The tea-leaves harness (MIT licensed, by cleak — github.com/cleak/tea-leaves) provides runtime inspection, automated validation, and linting tools purpose-built for AI-assisted Godot development. Integrating it enables Claude Code to verify changes automatically and catch broken assets before they ship.

## User Story

As a developer using Claude Code, I want runtime inspection, validation, and automated quality gates so that AI can verify changes automatically and broken assets are caught before they ship.
